# saucedemo-ui-automation

UI automation framework for testing the SauceDemo e-commerce web app using Selenium, TestNG, and Allure, following Page Object Model best practices.

## Key Technologies and Tools

- **Java 21**
- **Selenium Manager**
- **TestNG**
- **Allure Reporting**
- **Maven**
- **Lombok**
- **Retry Analyzer**
- **Screenshot**
- **Parallel execution**
- **GitHub Actions CI**
